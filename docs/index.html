    RUBY ‘pubmedid2pdf’ Interactions¶

    SHIP._process_line(line)[source]¶

    Function to process IDs in ruby script If the download fails, the ID is returned to a list of failed IDs

    SHIP.run_id_ruby(file_path, kickback_path, num_threads=10)[source]¶

    Run ruby script on list of ids at file_path and output failed ids The IDs are taken in chunks and run in parallel

    SHIP.run_id_ruby_mp(file_path, kickback_path, num_threads=10)[source]¶

    Multiprocessing version of run_id_ruby - uses processes which opens multiple windows, not neccesary for ruby function

    JSON Parsing Functions¶

    SHIP.run_json_folder(json_path, exclude_path, char_path, bkup_csv_path, csv_out_path)[source]¶

    Creates an SQLite DataBase in memory Adds the data of the JSON files to the table Removes the unwanted sections and only keeps the authors and materials section Splits the entries into 3000 character chunks If the header reads as ‘null’ tries to find a header using capital letters Deletes sections based on their digit to alphabetic ratios

    SHIP.partition_jsons(json_dir, partition_dir, json_per_folder)[source]¶

    Splits the JSONs into multiple folders

    SHIP.run_partition_folders(part_folder_dir, out_csv_dir, exclude_path, char_path, limit=-1)[source]¶

    Passes each partioned JSON folder through ‘run_json_folder’

    CSV Parsing Functions¶

    SHIP.get_pmedid_csv(csv_file_path, output_path)[source]¶

    Make list of PubMed ids from PubMed csv

    Take PubMed csv at csv_file_path. Make list of entries with PMCID format “PMCID+PUBMEDID” at pmc_path. Make list of entries with no PMCID at nopmc_path.

    SHIP.csv_add_pcmid(csv_file_path, pmed_id_path, output_path)[source]¶

    Add PMCID to PUBMEDIDs in pmed_id_path txt. Output results in format “PMCID+PUBMEDID” to output_path.
